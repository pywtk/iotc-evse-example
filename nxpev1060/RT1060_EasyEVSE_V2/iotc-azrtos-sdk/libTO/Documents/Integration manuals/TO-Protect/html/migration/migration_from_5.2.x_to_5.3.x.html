TO library migration guide from 5.2.x to 5.3.xÂ¶

The following changes are to be taken into account to update from 5.2.x to 5.3.x.

Following certificate personalization functions has been deprecated:

  • TOSE_perso_cert_set_kpub()

  • TOSE_perso_cert_set_subject_cn()

  • TOSE_perso_cert_set_signature()

These values are now parsed in X509 personalized with TOSE_perso_cert_set_x509().

TOSE_perso_system_set_serial_number() is also useless when using TOSE_perso_cert_set_x509().
